## Langue
Tout le code, les commentaires et les messages de commit sont en français.

## Stack
Backend en TypeScript, compilé en CommonJS par `tsc` (sources dans `backend/`, sortie dans `dist/`).

## Commandes
- `npm run build` : compile le TypeScript (backend/ vers dist/)
- `npm start` : démarre le serveur compilé (`dist/server.js`, port 3000)
- `npm run dev` : démarre le serveur en TypeScript directement (ts-node), pour le développement
- `npm run lint` : analyse le code avec ESLint
